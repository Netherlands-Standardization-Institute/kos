Abstract

The NEN Knowledge Organization System describes a data model for standardisation bodies based on SKOS and Dublin Core. It consists of:

  • the International Classification for Standards, see [ICS],
  • a catalog with standard documents,
  • and a controlled vocabulary with terms and definitions.

The SKOS data model views this knowledge organization system as a concept scheme comprising a set of concepts. These SKOS concept schemes and SKOS concepts are identified by URIs, enabling anyone to refer to ICS classes, documents and terms and definitions unambiguously from any context, and making them a part of the World Wide Web.

This document is intended for readers who are involved in the design and implementation of information systems, and who already have a good understanding of Semantic Web technology, especially RDF, OWL and SKOS, see the [SKOS-REFERENCE].

Introduction

The NEN Knowledge Organization System aims to create a datamodel for normalized terms and definitions that can be easily published using Skosmos. Skosmos is an open source web-based SKOS browser and publishing tool, see [SKOSMOS]. It provides users with access to controlled vocabularies for indexing, information retrieval and vocabulary development.

Note that other ontologies exist for describing aspects of the publishing domain as well as the structure and elements of standards. We opted for a simple datamodel that enables standards bodies to make a first step in bringing standards to the semantic web.
